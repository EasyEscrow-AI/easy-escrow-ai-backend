Execute: solana-keygen new -o target/deploy/escrow-keypair.json --force

After generation:

- Verify the keypair file exists at target/deploy/escrow-keypair.json
- Store the program ID for use in subsequent tasks

Expected output: A new keypair file and a base58-encoded public key
