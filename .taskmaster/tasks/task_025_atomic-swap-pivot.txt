## 1. Create getTreeAuthority helper function for PDA derivation [done]
### Dependencies: None
### Description: Implement a helper function to derive tree authority Program Derived Addresses (PDAs) for compressed NFTs
### Details:
Create getTreeAuthority() function in transactionBuilder.ts that derives the tree authority PDA using the merkle tree public key and the bubblegum program ID. Use PublicKey.findProgramAddressSync() with appropriate seeds.

### Dependencies: 25.1

### Details:

## 3. Add cNFT-specific accounts for compressed NFT swaps [done]
### Dependencies: 25.1, 25.2
### Description: Add the required accounts for compressed NFT operations including merkle tree, tree authority, and program accounts
