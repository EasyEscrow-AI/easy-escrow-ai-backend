2. **Feature Flag Utility**: Create src/utils/featureFlags.ts with:

   - Centralized feature flag checking for consistency

3. **Update JITO Entry Points** - check flag before executing:

   - src/services/jito-region-router.ts - region routing (bypass when disabled)

   - Use standard sendTransaction() for single txs
   - Log warning that JITO is disabled
   - Return gracefully without throwing

5. **Preserve All Code**: No deletions - wrap JITO paths in conditionals

6. **Configuration Updates**:
   - Update .env.example with ENABLE_JITO_BUNDLES=false
   - Update .env.staging template
   - Update .env.production template
   - Document in README or deployment docs
