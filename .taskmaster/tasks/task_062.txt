**2. Troubleshooting Guide (docs/PROGRAM_ID_TROUBLESHOOTING.md)**
Document common gotchas and solutions:
- **Stale IDL Issue**: When backend uses old IDL after program changes
  - Symptoms: Transaction failures, instruction not found errors
  - Solution: Copy fresh IDL from target/idl/ and restart services
- **Wrong Runtime Address**: Mismatched program IDs in different configs
  - Symptoms: Program not found errors, deployment failures
  - Solution: Verify all .env files, Anchor.toml, and declare_id match
- **Accidentally Changed Program Keypair**: Lost or corrupted keypair
  - Symptoms: Cannot deploy to existing program address
  - Solution: Restore from backup in temp/ directory
