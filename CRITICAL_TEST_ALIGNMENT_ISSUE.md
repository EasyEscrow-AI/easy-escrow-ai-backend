**Two Different Program Interfaces Exist:**

1. **What's Deployed on Devnet** (Program ID: `7dVEyFFeMzAT3oUpyvXwchGfPQDuXHdQv5tyfDBztKuV`)

### Anchor Type Resolution

Anchor v0.32+ automatically:
1. Derives PDA accounts from seeds
2. Converts snake_case → camelCase
3. Resolves program IDs
4. Creates strong types

Tests must NOT manually specify PDAs that Anchor derives.
